### 🚀 Sobre mim

Atualmente estou no **Apple Developer Academy | PUC-Rio**, focado em me tornar **iOS Developer / Software Engineer** — aprendendo Swift, SwiftUI e boas práticas de engenharia de software aplicadas ao ecossistema Apple. 
Colaboro atualmente com o GameLab UERJ, participando do desenvolvimento de jogos em Godot.  
Também participo de um projeto de **Iniciação à Docência**, produzindo cursos e vídeo-aulas sobre desenvolvimento de jogos com **Godot Engine**.

---

### Background

Antes de migrar o foco para iOS, passei um bom tempo construindo sistemas, experiência que hoje sustenta minha forma de pensar arquitetura, mesmo em contextos diferentes:

- **Back-end web em produção** - Sistemas em Laravel/PHP, incluindo integração de pagamentos (checkout transparente, Pix e cartão).
- **Bancos de dados relacionais** - modelagem e stored procedures em SQL Server, com foco em integridade e consistência de dados.
- **CloudKit / iCloud** - persistência e sincronização de dados entre dispositivos em apps iOS, usando o banco de dados do CloudKit.
- **Segurança e distribuição de software** - launcher desktop (WPF/.NET) com verificação de integridade via SHA-256, pipeline de criptografia AES e sistema de autenticação anti-cheat para servidores de jogos.

Esse histórico é o que me atrai para **Software Engineering** como carreira: gosto de entender o sistema como um todo, não só a camada de interface.
